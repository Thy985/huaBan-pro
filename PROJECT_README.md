# 像素画编辑器 - 重构版

## 项目结构

### 核心模块

- `huaBan.h` - 项目核心头文件，定义常量、枚举、数据结构
- `canvas.h/cpp` - 画布模块，负责像素数据管理、绘制、缩放、平移
- `tools.h/cpp` - 工具模块，实现画笔、橡皮擦、填充等绘图工具
- `history.h/cpp` - 历史记录模块，实现撤销/重做功能
- `ui.h/cpp` - 用户界面模块，负责按钮、调色板、信息显示
- `fileio.h/cpp` - 文件IO模块，负责图像的导入和导出
- `main.cpp` - 主程序，整合所有模块，提供完整应用

### 原始文件

- `main_old.cpp` - 原始的单文件实现（备份）
- `hanShuo.cpp` - 原始文件

## 架构改进

### 1. 模块化设计
- 将1600+行的单文件拆分为7个独立模块
- 每个模块职责单一，便于维护和扩展

### 2. 面向对象
- 使用类封装功能（Canvas、Tools、History、UI、FileIO）
- 消除了全局变量的依赖
- PixelEditor类整合所有模块

### 3. 代码优化
- 更好的代码组织和可读性
- 简化的函数接口
- 更容易进行单元测试

## 功能特性

- ✅ 32x32 像素画布编辑
- ✅ 画笔、橡皮擦、油漆桶工具
- ✅ 可调画笔大小（1x/2x/3x）
- ✅ 16色预设调色板 + 最近使用颜色
- ✅ 撤销/重做功能（最大50步）
- ✅ 画布缩放（100%-400%）
- ✅ 画布平移（Ctrl+拖动）
- ✅ 网格显示控制
- ✅ 图像导入/导出（PNG/BMP）
- ✅ 快捷键支持（Ctrl+Z、Ctrl+Y、Ctrl+S）

## 编译说明

### 使用 Visual Studio
1. 打开 `huaBan-pro.sln` 解决方案文件
2. 确保已安装 EasyX 图形库
3. 编译并运行项目

### 添加新源文件
确保在项目中包含以下新文件：
- canvas.cpp
- tools.cpp
- history.cpp
- ui.cpp
- fileio.cpp
