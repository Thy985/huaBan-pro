# 像素画编辑器 - 重构版

## 项目结构

### 核心模块

- `huaBan.h` - 项目核心头文件，定义常量、枚举、数据结构
- `canvas.h/cpp` - 画布模块，负责像素数据管理、绘制、缩放、平移
- `tools.h/cpp` - 工具模块，实现画笔、橡皮擦、填充等绘图工具
- `history.h/cpp` - 历史记录模块，实现撤销/重做功能
- `ui.h/cpp` - 用户界面模块，负责按钮、调色板、信息显示
- `fileio.h/cpp` - 文件IO模块，负责图像的导入和导出
- `main.cpp` - 主程序，整合所有模块，提供完整应用

### 新增功能模块

- `layer.h/cpp` - 图层系统模块，支持多图层编辑
- `filter.h/cpp` - 滤镜模块，提供图像处理滤镜
- `animation.h/cpp` - 动画模块，支持帧动画编辑
- `plugin.h` - 插件接口定义，支持第三方扩展
- `plugin_manager.h/cpp` - 插件管理器，动态加载和管理插件

### 原始文件

- `main_old.cpp` - 原始的单文件实现（备份）
- `hanShuo.cpp` - 原始文件

## 架构改进

### 1. 模块化设计
- 将1600+行的单文件拆分为多个独立模块
- 每个模块职责单一，便于维护和扩展

### 2. 面向对象
- 使用类封装功能（Canvas、Tools、History、UI、FileIO、LayerManager等）
- 消除了全局变量的依赖
- PixelEditor类整合所有模块

### 3. 代码优化
- 更好的代码组织和可读性
- 简化的函数接口
- 更容易进行单元测试
