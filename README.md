# 画版Pro - 像素艺术编辑器

![Build Status](https://github.com/Thy985/huaBan-pro/actions/workflows/build.yml/badge.svg)

## 项目介绍
画版Pro是一个功能强大的像素艺术编辑器，专为Windows系统设计，使用EasyX图形库实现图形界面。项目采用模块化、面向对象的设计理念，具有良好的可扩展性和可维护性。

## 快速开始

### 方法一：使用自动构建脚本（推荐）

1. **确保已安装Python**
   - 要求Python 3.6或更高版本
   - 可以从 [Python官网](https://www.python.org/downloads/) 下载安装

2. **运行构建脚本**
   - 在项目根目录下双击运行 `build.py`
   - 或在命令行中执行：
     ```bash
     python build.py
     ```

3. **脚本会自动完成以下操作**
   - 检查系统环境和Python版本
   - 下载并安装 CMake（如果需要）
   - 下载并安装 EasyX 图形库（如果需要）
   - 检查C++编译器（Visual C++或MinGW）
   - 构建项目
   - 运行应用

### 方法二：使用CMake手动构建

1. **安装依赖**
   - CMake 3.10或更高版本
   - EasyX 图形库
   - C++编译器（Visual C++或MinGW）

2. **构建步骤**
   ```bash
   mkdir build
   cd build
   cmake ..
   cmake --build . --config Release
   ```

3. **运行应用**
   - 构建完成后，可执行文件位于 `build/bin` 目录

- ✅ 可调画笔大小（1x/2x/3x）
- ✅ 16色预设调色板 + 最近使用颜色
- ✅ 撤销/重做功能（最大50步）
- ✅ 画布缩放（100%-400%）
- ✅ 画布平移（Ctrl+拖动）
- ✅ 网格显示控制
- ✅ 图像导入/导出（PNG/BMP）
- ✅ 快捷键支持（Ctrl+Z、Ctrl+Y、Ctrl+S）

## 系统要求

- Windows 7/8/10/11
- Python 3.6+（仅用于构建脚本）
- C++编译器（Visual C++或MinGW）
- 至少2GB内存
- 支持DirectX的显卡

## 故障排除

### 常见问题

1. **缺少管理员权限**
   - 构建脚本需要管理员权限来安装依赖
   - 系统会自动提示以管理员权限重新运行

2. **缺少C++编译器**
   - 脚本会检测编译器并提供安装建议
   - 推荐安装 Visual Studio Build Tools 或 MinGW-w64

3. **网络下载失败**
   - 确保网络连接正常
   - 检查防火墙设置
   - 手动下载依赖并安装

4. **构建失败**
   - 查看 `build.log` 文件获取详细错误信息
   - 确保所有依赖已正确安装
   - 尝试清理 `build` 目录后重新构建
